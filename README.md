# Bottini Fuel — Website Clone

A front-end recreation of the Bottini Fuel corporate website, built for
responsive design and CSS architecture practice.

## Stack

HTML5 · CSS3 · JavaScript (vanilla)

## Running it

No build step — open `index.html` in a browser, or run a local server:

```bash
python -m http.server 8000
```

## What's in it

- Responsive layout built with CSS Grid and Flexbox
- Custom properties for colors and spacing
- Sticky nav, hero section, services cards, CTA blocks, footer
- Basic scroll behavior in JS

## Notes

Built as a pixel-matching exercise. Color palette and typography pulled
from the original site. No frameworks or dependencies.
